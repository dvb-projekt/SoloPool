# Contributing to SoloPool

This is a community fork of BlackHole Pool. Contributions are welcome —
please follow the upstream license (see `LICENSE`) for any code you submit.

## Bug reports

Open an issue with:
- Pool version / commit
- Miner model + firmware
- Stratum logs and the failing behavior

## Miner compatibility reports

We track compatibility per miner model. If you've tested a model that
isn't documented yet, please open an issue with the **Miner compatibility**
template.

## Pull requests

- Keep changes focused — one topic per PR
- Update release notes for user-visible changes
- By contributing you agree your changes are released under the
  repository's existing license

## Upstream sync

This fork is periodically synced with upstream BlackHole Pool.
If your change makes sense upstream too, please consider sending it there.
